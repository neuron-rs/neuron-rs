# Neuron

## Overview
Neuron is an actor system for building reactive, concurrent, and resilient distributed services, for the Rust programming language.

* [Website](https://neuron.rs)
* [API Documentation](https://docs.rs/neuron-rs/)
* [Examples](https://github.com/neuron-rs/neuron-examples)
